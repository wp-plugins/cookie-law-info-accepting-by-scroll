== Description ==

This plugin integrates with the Cookie Law Info plugin by Richard Ashby and permits acceptance of the use of cookies on your device by scrolling the page.
Also reloads the page after clicking on the "accept" button.

== Installation ==

1. Upload `cookie_law_info_accept_scroll` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
